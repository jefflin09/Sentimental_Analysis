# Sentiment Analysis using ML & Deep Learning

This project performs sentiment analysis on movie reviews using:
- Machine Learning (TF-IDF + Logistic Regression)
- Deep Learning (LSTM Neural Network)

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- TensorFlow / Keras
- Kaggle Dataset (IMDB Movie Reviews)

## Features
- Trains ML and DL models
- Compares accuracy
- Predicts sentiment for user input

## How to Run
1. Install dependencies
2. Run `sentiment.py`
3. Enter a sentence to predict sentiment
